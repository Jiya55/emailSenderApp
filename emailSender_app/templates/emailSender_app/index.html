This website is an integration of my Python CLI/GUI. This app was created using HTML, CSS, Python, and Django. The user of this web application must click the "Go To Email Sender" button. This will open a form where you must fill out the sender email, sender app password (information on where to get that is supplied under the form), receiver email (at this time, only one receiver may be added), subject, and body. After that, you must push the email send button. It can take a while, so please only press it once and be patient. Pressing the button multiple times will send the email multiple times, spamming the receiver's inbox. You will be taken to a thank you page.
